# Webhook Settings

## Overview

**`Webhook`** is based on **`Project`**  page that allows you the following actions:

* **Set up the Alert Webhook.** 
* **Set up Notification Policy** : View, and Edit 
  * Notification Policy
  * Auto Recovery
  * Event Rule
* **Escalation Policy**
